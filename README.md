# 📰 Fake News Detection System using Passive Aggressive Classifier

An AI-powered machine learning project that detects fake news articles using Natural Language Processing (NLP) techniques and the Passive Aggressive Classifier algorithm.

---

## 📌 Project Overview

The project aims to identify whether a news article is real or fake by analyzing textual content using machine learning and NLP techniques. It helps reduce misinformation and improve content credibility.

---

## 🚀 Features

* Detects fake and real news articles.
* Uses NLP preprocessing techniques.
* Implements TF-IDF vectorization.
* Uses Passive Aggressive Classifier for prediction.
* Provides high prediction accuracy.
* Supports real-time news classification.

## 🤖 Machine Learning Model

### Passive Aggressive Classifier (PAC)

* Vectorization Method: TF-IDF
* Vocabulary Size: 10,000 Features
* Classification Type: Binary Classification
